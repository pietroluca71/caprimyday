Badge — small label for status, category, or metadata.

```jsx
<Badge variant="rosa">Featured</Badge>
<Badge variant="neutral" size="md">Confirmed</Badge>
<Badge variant="success">Paid</Badge>
```

### Variants
- `rosa` — pink tint, default for brand tags
- `neutral` — grey, for general metadata
- `success` / `warning` / `error` — status indicators

### Props
- `size`: `sm` | `md`
